# Actividad M4

En esta actividad M4, se realizaron varias mejoras significativas en el proyecto en comparación con la entrega anterior.

Algunos cambios clave:

Refactorización y Modularización del Código:
Refactorización del código para mejorar la estructura y legibilidad. El código se dividió en módulos más pequeños y utilizando funciones y métodos bien definidos para abstraer la lógica.

Implementación del Patrón DAO: Se introdujo el Patrón DAO (Data Access Object) para gestionar la interacción con la base de datos. Esto ha mejorado la separación de preocupaciones y la escalabilidad del proyecto.
